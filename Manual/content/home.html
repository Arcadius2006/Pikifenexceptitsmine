Welcome!

Welcome to Pikifen's user manual!

This manual will teach you how to play and how to make your own content.

How to play

To play, just choose the "Play" option from the main menu and pick an area. The following pages might help with more specific topics:

  • Controls
  • Options - If you need to fine-tune the options, this page will explain them in detail.
  • Tips

How to make

You can make any content to be run on the engine, and the ways to do so vary, but are in general made to be easy and intuitive. Enemies, Pikmin types, and other such creations can be freely made, but new game mechanics and logic cannot. This also includes enemies, areas, etc. that make use of any new mechanic. The following pages lead to tutorials on how to make different content for the engine:

  • Quick start guide – for users who need help understanding the basics, but believe they can figure out the rest on their own.
  • In-depth guides:
    • Text file – text files on the engine use a specific format. This simple page explains how it works, and should be read before using any other tutorial.
    • Making areas
    • Making new object types
    • Making animations
    • Object scripting
    • Making hazards
    • Making spray types
    • Making status effects
    • Making particle generators
    • Making new weather
    • Misc. configurations
    • Misc. features
  • Maker toolkit – tools to use to help you make and debug content.
  • Object category pages – information on the requirements and such for objects of every category.

Graphics and audio can also be edited freely. Although some images and sounds need to have specific names for the engine to find them, others can use any name at all, so long as the scripts, areas, etc. that use them call them by the right name.

Most user-made mistakes will be caught by the engine and reported in the file /User_data/Error_log.txt.
