GUI

All menus and HUDs in Pikifen can be customized. Most of them only allow you to change the position and size of the different elements, but some may allow for more in-depth customizations. Each menu or HUD has its own text file, located in Game_data/GUI.

The following sections of this page explain the different menus and HUDs that can be customized. Each section contains the name of all GUI items in that menu/HUD. In the equivalent text file, you can edit the positions block to change the position and size of the items. The format of each line in the positions block is: item = x y w h, where item is its name, x and y are the position, and w and h are the sizes. Note that both the coordinates and sizes are in window percentages. This means an X of 50 will be horizontally exactly in the middle of the window, a H of 30 will make the item take 30% of the window's height, etc.

Main menu

  • Items:
    • play: Play button.
    • options: Options button.
    • animation_editor: Animation editor button.
    • area_editor: Area editor button.
    • exit: Exit button.

Area menu

  • Items:
    • back: Back button.
    • pick_text: Instructions for the player to pick an area.
    • list: List of areas.
    • list_scroll: Scrollbar for the list.
